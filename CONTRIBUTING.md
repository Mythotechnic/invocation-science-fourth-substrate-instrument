# Contributing

This repository is part of the Invocation Science® research corpus.

At this time, external pull requests are only acceptable if they:

- Preserve full attribution to Invocation Science® and the original author.
- Do not attempt to rename, rebrand, or generalize the instrument as a new,
  independent framework or "field".
- Clearly document how proposed changes relate to the AIA / Drift Engine /
  Fourth Substrate manuscripts.

If in doubt, please open an issue describing your intended changes and how they
respect Invocation Science® field sovereignty and licensing.
